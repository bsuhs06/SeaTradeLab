# New Project: Sea Trade Lab

I've been working on this project for some time and decided it was worth documenting the process. This blog will cover findings, methodology, technical decisions, and lessons learned along the way.

First, a note on intent: this project is not political. It isn't designed to support or oppose any state, organization, or individual. The purpose is strictly analytical and educational.

![Sea Trade Lab]({{ '/assets/images/posts/GenericScreen.png' | relative_url }})

The focus is on understanding global maritime trade patterns using publicly available data — specifically, how shipping activity adapts during periods of geopolitical tension and economic disruption. Trade networks are remarkably responsive to external pressure, and shifts in routing, ownership structures, and operational behavior are often observable through open-source AIS data.

Maritime commerce underpins the global economy in ways that rarely get public attention. As geopolitical dynamics continue to evolve, the maritime domain is likely to become an increasingly important area of economic and strategic activity. This project aims to explore those dynamics from a data-driven perspective.

What I'm building is an open-source analytical framework for examining vessel movements, fleet behavior, and anomalies in shipping patterns. All analysis relies on legally obtained, publicly accessible information. Any published findings will be appropriately delayed or aggregated to avoid real-time operational sensitivity.

A secondary objective is to explore how modern analytical tools — including AI — can enhance open-source intelligence workflows. The question I'm most interested in is whether an individual researcher with modest technical resources can use freely available data and APIs to surface meaningful patterns in global shipping.

If the answer is yes, that has real implications for transparency, investigative journalism, and independent research. At the same time, this work will respect privacy, legal boundaries, and ethical considerations in how data is collected, analyzed, and shared.

## Disclaimer

This project uses publicly available data for research and educational purposes only. It does not provide real-time tracking.
